In Benchmark 1, the main result is not that Gemini cannot compute an RC cutoff. It can. The result is that CiTT connects the same calculation to a model, probe placement, focus-linked teaching, and unit-mistake evidence. In Benchmark 2, CiTT demonstrates structural model evidence and honest symbolic handling. In Benchmark 3, CiTT demonstrates that model grounding can expose failure modes such as saturation and non-settling instead of smoothing them over in prose.

The release package was also tested operationally. The \texttt{.mltbx} installed, \texttt{which citt} resolved to the MATLAB Add-Ons path, \texttt{citt.checkSetup} ran, the GUI launched, and the post-package examples script reproduced all three example workflows. These tests support reproducibility of the software package, not universal correctness of every generated model.
