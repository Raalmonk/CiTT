# LLM Baseline Prompt

An ECG acquisition front end uses a first-order RC low-pass filter before a 500 Hz ADC. The circuit uses R = 39.8 kOhm and C = 100 nF. The input contains a 5 Hz ECG-like component and 60 Hz interference.

Compute the cutoff frequency, attenuation at 60 Hz, attenuation at the Nyquist frequency, identify where the output voltage should be probed in a Simscape/Simulink model, and diagnose a lab mistake where the student accidentally used 100 uF instead of 100 nF.

Explain your reasoning clearly. Do not assume access to Simscape or executable model evidence.
